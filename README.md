## 🚀 How to use

```bash
# Install dependencies
yarn install
# Start the Server
yarn android # for android
yarn ios # for ios
yarn web # for web
yarn dev # tunnel
yarn start # lan
```

### ✨ Features

* User authentication with Firebase.
* Create, edit, delete, and mark tasks as done.
* Real-time updates with Firestore.
* Offline support.
